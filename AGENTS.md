# AGENTS.md

## Project guidance
- This repository is a private-preview Astro site using Tailwind CSS and pnpm.
- Keep sensitive PDF, Word, Excel, and complete video assets out of `public/`.
- Preserve search blocking unless the project owner explicitly asks to publish the site: `noindex, nofollow, noarchive` and `public/robots.txt` with `Disallow: /`.
- Run `pnpm build` before opening a pull request.
